Code you can actually run

Every script in the repo builds on a clean machine. These renders on this page are produced headlessly from the scripts below — not pre-baked images.

Parametric bracket — OpenSCAD

openscad -o bracket.stl \
  scripts/openscad/parametric_bracket.scad

Open the Customizer to drive length, depth, thickness, hole count, fillet, and rib from sliders.

Involute gear — OpenSCAD

openscad -o gear.stl \
  -D 'teeth=24' -D 'module_mm=2' \
  scripts/openscad/involute_spur_gear.scad

True involute flanks, correct base / pitch / root circles, helical if you set a twist angle.

FreeCAD macro — native solid + STEP

freecadcmd \
  scripts/freecad-python/generate_bracket.py

Builds the same bracket natively in FreeCAD, exports STEP and STL to ~/cad-portfolio-exports/.

IFC wall audit — IfcOpenShell

python scripts/blender-bim/ifc_wall_audit.py \
  building.ifc

Markdown report: declared vs. geometric length / height, Pset coverage, material assignment.
