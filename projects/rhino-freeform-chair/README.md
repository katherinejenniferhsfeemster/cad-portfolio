# Freeform Lounge Chair

A class-A surfaced lounge chair, modeled in **Rhino 3D** with **Grasshopper** driving the cushion parametrics, finished with a subD-to-NURBS workflow.

**Tools:** Rhino 3D, Grasshopper, Blender (for retopo and UVs), KeyShot / Cycles for renders.

## What this demonstrates

- **Class-A surfacing** — G2 continuity across the shell, zero-deviation curvature comb reviews.
- **SubD → NURBS** — hero form blocked in Rhino SubD, converted and re-lofted as NURBS for manufacturing.
- **Grasshopper parametrics** — cushion thickness, seat angle, and stitch line count driven by four sliders.
- **Downstream outputs** — tessellated mesh for renders, NURBS STEP for the mold shop, lightweight glTF for web preview.
